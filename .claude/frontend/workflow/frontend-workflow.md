# frontend-workflow.md

Frontend Development Workflow:

1. Read architecture skills
2. Review feature structure
3. Create composables
4. Create reusable components
5. Create route pages
6. Add tests
7. Refactor
8. Verify responsiveness

Checklist:

* responsive UI
* loading states
* accessibility
* reusable logic
* proper typing
* no console errors

Important:
Frontend quality directly affects learning experience.

# frontend-performance.md

Performance Rules:

* lazy-load routes
* split bundles by feature
* avoid unnecessary re-renders
* debounce expensive inputs

Lists:

* paginate large datasets
* optimize flashcard rendering
* avoid rendering hidden UI

Reactivity:

* avoid deep watchers
* avoid unnecessary computed chains

Assets:

* optimize images
* lazy-load media
* compress uploads

Important:
Fast interactions are critical for learning apps.

# design-system.md

Design Philosophy:

* minimal
* calm
* distraction-free
* learning-focused

Inspired By:

* Duolingo
* Anki
* Quizlet
* Notion

Typography:

* large readable text
* clear hierarchy
* accessible contrast

Spacing:

* consistent spacing scale
* generous whitespace
* avoid clutter

Components:

* reusable UI system
* consistent buttons
* consistent form styles

Color Rules:

* avoid excessive colors
* use calm palettes
* prioritize readability

Important:
Consistency is more important than decoration.

# animations.md

Animation Philosophy:

* subtle
* smooth
* purposeful
* never distracting

Rules:

* animations should improve UX
* avoid excessive motion
* keep durations short
* use consistent easing

Flashcard Rules:

* smooth card flip animations
* clear front/back distinction
* responsive touch interactions

Performance:

* prefer transform animations
* avoid expensive layout changes

Important:
Animations should support learning focus.

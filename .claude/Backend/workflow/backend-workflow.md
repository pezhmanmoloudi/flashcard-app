# backend-workflow.md

Backend Development Workflow:

1. Read architecture documents
2. Create models/migrations
3. Create services
4. Create API endpoints
5. Add serializers
6. Add tests
7. Refactor
8. Optimize queries

Checklist:

* proper indexes
* authorization checks
* request specs
* service objects
* no business logic in controllers

Important:
Backend architecture must remain maintainable and scalable.
